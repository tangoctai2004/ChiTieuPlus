CƠ SỞ DỮ LIỆU (CORE DATA SCHEMA)

1. Entity: CATEGORY (Danh mục)
- id: UUID (Khóa chính)
- name: String (Tên danh mục)
- type: String (Loại: "income" hoặc "expense")
- icon: String (Tên SF Symbol)
- color: String (Mã màu Hex)
- createAt: Date (Ngày tạo)
- updateAt: Date (Ngày cập nhật)
* Quan hệ: 1-N với Transaction (Một danh mục có nhiều giao dịch).

2. Entity: TRANSACTION (Giao dịch)
- id: UUID (Khóa chính)
- title: String (Tiêu đề/Mô tả ngắn)
- amount: Double (Số tiền)
- date: Date (Ngày giao dịch)
- type: String (Loại: "income" hoặc "expense")
- note: String (Ghi chú chi tiết)
- categoryID: UUID (Khóa ngoại - Tham chiếu đến Category)
- createAt: Date (Ngày tạo)
- updateAt: Date (Ngày cập nhật)
* Quan hệ: N-1 với Category (Mỗi giao dịch thuộc về 1 danh mục).

3. Entity: BUDGET (Ngân sách)
- id: UUID (Khóa chính)
- amount: Double (Số tiền hạn mức)
- period: String (Kỳ hạn: "monthly", "weekly", "yearly")
- startDate: Date (Ngày bắt đầu áp dụng)
- warningThresholds: String (Các mốc cảnh báo, VD: "80,90")
- rolloverEnabled: Bool (Cho phép chuyển số dư sang kỳ sau)
- isActive: Bool (Trạng thái hoạt động)
- categoryID: UUID (Khóa ngoại - Tham chiếu đến Category)
* Logic: Thiết lập giới hạn chi tiêu cho một danh mục cụ thể.

4. Entity: SAVINGS_GOAL (Mục tiêu tiết kiệm)
- id: UUID (Khóa chính)
- title: String (Tên mục tiêu)
- targetAmount: Double (Số tiền mục tiêu)
- currentAmount: Double (Số tiền hiện có)
- startDate: Date (Ngày bắt đầu)
- targetDate: Date (Ngày dự kiến hoàn thành)
- isCompleted: Bool (Trạng thái hoàn thành)
- iconName: String (Icon đại diện)
- color: String (Màu sắc)
* Logic: Hoạt động độc lập, người dùng nạp tiền vào mục tiêu.

5. Entity: RECURRING_TRANSACTION (Giao dịch định kỳ)
- id: UUID (Khóa chính)
- title: String (Tên giao dịch)
- amount: Double (Số tiền)
- frequency: String (Tần suất: "daily", "weekly", "monthly", "yearly")
- startDate: Date (Ngày bắt đầu)
- endDate: Date? (Ngày kết thúc - Tùy chọn)
- nextDueDate: Date (Ngày đến hạn tiếp theo)
- isActive: Bool (Trạng thái hoạt động)
- categoryID: UUID (Khóa ngoại - Tham chiếu đến Category)
* Logic: Hệ thống tự động tạo Transaction mới khi đến nextDueDate.
